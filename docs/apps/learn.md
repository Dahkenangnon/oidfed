# @oidfed/learn

An interactive course on OpenID Federation 1.0 — 15 lessons from first principles to federation topology design, with hands-on exercises and spec-accurate references.

## Role

Structured course at [learn.oidfed.com](https://learn.oidfed.com) — walks readers through the specification from core concepts to federation topology design across 15 interactive lessons.

## Build

```bash
pnpm --filter @oidfed/learn build
# → apps/learn/build/  (static site, prerendered)
```

## Deploy Target

Live → `learn.oidfed.com`
