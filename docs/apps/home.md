# @oidfed/home

The official home of @oidfed — the complete OpenID Federation 1.0 implementation for JavaScript, runtime-agnostic, spec-compliant, built on Web API standards.

## Role

Landing page at [oidfed.com](https://oidfed.com) — introduces the project and links to the explorer, learn app, and GitHub repository.

## Build

```bash
pnpm --filter @oidfed/home build
# → apps/home/build/  (static site, prerendered)
```

## Deploy Target

Live → `oidfed.com`
